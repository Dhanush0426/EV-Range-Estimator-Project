# Electric Vehicle Range Prediction using Machine Learning

## 🔍 Problem Statement
Predict the driving range (in kilometers) of electric vehicles using features such as battery capacity, efficiency, and top speed.

## 📊 Dataset
Dataset: *Electric Vehicle Specifications Dataset 2025*  
Source: [Kaggle](https://www.kaggle.com)  
File used: `electric_vehicles_spec_2025.csv.csv`

- Checked missing values and data types  
- Created simple scatterplots for basic EDA
